# Plugin reference

The economy is built from several plugins working together. Each page below opens with
a short note on **what the plugin is and why it exists**, then lists everything it does
— commands, arguments, examples, and the roles or permissions you need.

- **[Treasury](/docs/reference/treasury)** — the server's money system: paying,
  balances, transactions, government accounts, and fines.
- **[Business](/docs/reference/business)** — companies (firms): staff, roles, accounts,
  money, and tax.
- **[ChestShop](/docs/reference/chestshop)** — automatic sign shops: the sign format and
  every command.
- **[Realty](/docs/reference/realty)** — land: buying, renting, offers, auctions, and
  plot management.

> [!NOTE]
> In the argument tables, `<angle brackets>` mean a required value and
> `[square brackets]` mean an optional one. Don't type the brackets themselves.
