# Economy documentation

Welcome! This is the player guide to the server economy. It explains how to **earn,
spend, and manage money** in-game — running a business, hiring staff, selling things
through ChestShops, and buying or renting property — and lists every command you'll
need along the way.

Everything here is about what you type **in-game**. If you just want to *look at* the
economy — balances, market prices, money flow — head back to the
[Economy Explorer](/) instead.

## New here? Start with the basics

You already have a balance the moment you join. The three commands you'll use most:

```text
/bal                 Check your balance
/pay <player> <amount>   Send money to someone
/transactions        See where your money went
```

> [!TIP]
> Type `/baltop` to see the richest players on the server.
